no subscript? Assume base 10

  • Each digit in a decimal number is multiplied by powers of the base, which is 10
  • The power is the place position, starting at 0 on the rightmost side
  • Multiply each digit by the base to the power of the place position
  • Add the products
  • This is the value of the number
  • Showing the number this way is termed expanded form

The base is 2 To convert the number to base 10, multiply each digit by powers of the base corresponding to the place position, then add.

  • Each digit in a binary number is multiplied by powers of the base, which is 2
  • The power is the place position, starting at 0 on the rightmost side, termed the LSB (least significant bit)
  • Multiply each digit by the base to the power of the place position
  • Add the products
  • This is the value of the number
  • Showing the number this way is termed expanded form
